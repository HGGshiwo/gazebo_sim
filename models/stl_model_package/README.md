# 外场演示动画模型 - STL版本

## 文件说明

本压缩包包含以下文件：

### 1. 外场演示动画.stl
- **文件大小**: 约20MB
- **格式**: STL (STereoLithography) 二进制格式
- **内容**: 完整的3D几何模型，包含多个组件如围墙、定位块、龙门吊等
- **用途**: 用于3D打印、CAD软件、仿真环境等

### 2. stl_model_world.sdf
- **格式**: SDF (Simulation Description Format) v1.6
- **内容**: Gazebo仿真世界配置文件
- **功能**: 定义模型在Gazebo中的位置、碰撞属性、视觉属性等
- **模型名称**: 外场演示动画模型_STL

### 3. model.config
- **格式**: XML配置文件
- **内容**: 模型元数据信息
- **包含**: 模型名称、版本、作者信息、描述等

## 使用方法

### 在Gazebo中使用
1. 解压本压缩包到任意目录
2. 使用命令启动Gazebo并加载模型：
   ```bash
   gazebo stl_model_world.sdf
   ```
3. 模型将自动加载到Gazebo仿真环境中

### 在其他3D软件中使用
- 直接使用`外场演示动画.stl`文件
- STL格式广泛支持于各种3D建模软件

## 技术规格

- **模型类型**: 静态模型
- **坐标系**: 右手坐标系
- **单位**: 米
- **几何精度**: 高
- **适用场景**: 工业仿真、3D可视化、机器人仿真

## 注意事项

- 模型为静态模型，不支持动态物理仿真
- STL格式不包含材质信息，模型将以默认材质显示
- 建议在具有足够内存的系统上加载（推荐4GB+）
